Ready-set-go

The ready-set-go process with three steps.

GET /rsg/index READ /rsg/index to get state, links and forms

200

rsg/index#READ-rsg-index-200
POST /rsg/index Create /rsg/{id} (EXISTS-rsg-id) and subordinate resources (NOT-EXISTS-ready-id...)

201

rsg/index#CREATE-POST-rsg-index-201
GET /rsg/{id} Read /rsg/{id} to get state, links and forms

200

rsg/index#READ-rsg-id-200
POST /rsg/{id} Reinitialise /rsg/{id} (EXISTS-rsg-id, NOT-EXISTS-ready-id...)

200

rsg/index#RPC-rsg-id-200
DELETE /rsg/{id} Delete /rsg/{id} (GONE-id) and subordinate resources for good

410

rsg/index#DELETE-rsg-id-410
